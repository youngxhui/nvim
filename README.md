# neovim 配置


neovim >= 0.10

采用了 blink 配置。


## 集成功能

开箱即用。

- Go
- Markdown
- React
- Rust
- Typescript

## 如何使用

将文件存放到 nvim 的配置文件夹下，例如 mac 的 `~/.config/nvim/` 下。

初次启动会安装 `Lazy` 。

再次启动会安装相关的插件。安装完插件后退出即可。

第三次启动的时候，运行 `:LspInstall all` 和 `TSIntall all` 安装所有的代码高亮

## codeium 使用

命令行输入 `Codeium Auth`，会打开浏览器进行授权，将授权码复制到 nvim 中。
